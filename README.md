# Protein Stability Prediction 🧬

This project predicts the stability of a protein using its amino acid sequence.

The system analyzes the sequence and calculates a stability score to classify whether the protein is stable or unstable.

---
## Features

- Accepts protein amino acid sequence input
- Calculates stability score
- Classifies protein as Stable or Unstable
- Displays sequence statistics (hydrophobic, charged, polar residues)

## How It Works
1. User enters a protein sequence.
2. The system validates amino acids (ACDEFGHIKLMNPQRSTVWY).
3. The program analyzes residue composition.
4. A stability score is calculated.
5. The protein is classified as Stable or Unstable.
